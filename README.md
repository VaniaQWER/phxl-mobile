# phxl-mobile

> phxl 移动端常用组件库
## Install

```bash
npm install --save phxl-mobile
```

## Usage

```tsx
import * as React from 'react'

import { A, B, C } from 'phxl-mobile'

class Example extends React.Component {
  render () {
    return (
      <A />
    )
  }
}
```
